FIND DETAILS OF POSITIONS

Overview

Returns the details of the positions along with the ID, Name and the Deeplink URL based on the provided search criteria.

Requires OAuth

No

Method

GET

URL

https://<hostname-api.sabacloud.com>/v1/positions?q=(:criteria_field%3D%3D:field_value)&count=:count&startPage=:startPage

Calling Options

Table 2. Calling OptionsCalling Options
Name Description Default Value Data Type Required?
criteria_field The search criteria i.e. the field name. status string Yes
field_value The search value for the specified search criteria. 100 string Yes
count The number of records per page. 10 string No
startPage The start page number for the list of records. 1 string No

Return Values

{
    "hasMoreResults": false,
    "itemsPerPage": 10,
    "startIndex": 1,
    "totalResults": 1,
    "facets": [],
    "results": [
        {
            "id": "postn000000000001302",
            "href": "https://{hostname}-api.sabacloud.com/v1/positions/postn000000000001302"
        }
    ]
}

Overview

Requires OAuth

No

Method

POST

URL

https://<hostname-api.sabacloud.com>/v1/positions/searchQuery?count=:count&startPage=:startPage

Calling Options

Table 3. Calling OptionsCalling Options
Name Description Sample Value Data Type Required?
conditions > name The search conditions.
You can use any field with "isSearchFilter": true.
Note: To find out the fields that have "isSearchFilter": true, invoke the "Get the Meta details" REST APIs.
You can use search operators such as:
  • =gt=
  • =ge=
  • =lt=
  • =le=
  • =eq=
  • =ne=
 string Yes
count The number of records per page. 10 string No
startPage The start page number for the list of records. 1 string No

{
    "conditions":[
        {
            "name":"status",
            "operator":"==",
            "value":"300"

        {
            "name":"positionTitle",
            "operator":"==",
            "value":"new position 001"

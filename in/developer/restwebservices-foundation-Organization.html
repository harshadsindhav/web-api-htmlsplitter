Overview

Requires OAuth

No

Method

URL

https://<hostname-api.sabacloud.com>/v1/organization?type=:type&count=:count&startPage=:startPage

Overview

Returns the details of the Internal/External Organizations along with the ID, Name and the Deeplink URL based on the provided search criteria.

Requires OAuth

No

Method

GET

URL

https://<hostname-api.sabacloud.com>/v1/organization?type=:type&q=(:criteria_field==:field_value)&count=:count&startPage=:startPage

Calling Options

Table 2. Calling OptionsCalling Options
Name Description Sample Value Data Type Required?
type The type of organization. internal enumerated Yes
criteria_field The search criteria i.e. the field name.
You can use any field with "isSearchFilter": true.
Note: To find out the fields that have "isSearchFilter": true, invoke the "Get the Meta details" REST APIs.
 name string Yes
field_value The search value for the specified search criteria. abc string Yes
count The number of records per page. 10 string No
startPage The start page number for the list of records. 1 string No

GET DETAILS OF AN ORGANIZATION

Overview

Returns complete information about an Organization based on the Organization's ID that is passed as a parameter value.

Requires OAuth

No

Method

GET

URL

https://<hostname-api.sabacloud.com>/v1/organization/:id
Note: Use ?includeassociation=true to retrieve the association details as well:

https://<hostname-api.sabacloud.com>/v1/organization/:id?includeassociation=true

URL (User-friendly)

You can use a user-friendly URL which accepts the number instead of the Organization's ID.
https://<hostname-api.sabacloud.com>/v1/organization/number%3D<OrgNo>?type=:type&includeassociation=true
Note: The type parameter is mandatory.

GET REPORT DEFINITION

Overview

Returns more detailed report definition which includes report definition information, filters, display attributes, chart information with chart attributes.

Requires OAuth

No

Method

GET

URL

https://<hostname-api.sabacloud.com>/v1/report/list/:reportId

Example:

https://<hostname-api.sabacloud.com>/v1/report/list/rptdf0000007904c03f0156a16c63e5008000

Calling Options

Table 1. Calling OptionsCalling Options
Name Description Default Value Data Type Required?
reportId Report ID.

Starts with: ‘rptdf’

 string Yes
audienceType Audience Type.

If not specified, the API will return the report definition detail with respect to ADMIN audience type.

If specified, the API will return the report definition detail if the report is shared with the specified audience type.

Refer: #ENUM for supported audience type.

string Yes

GET REPORT LIST

Overview

Returns a list of reports in which report as a service configuration is enabled (report specific config).It will mainly return the necessary details of the report and all the charts associated with that report.

Requires OAuth

No

Method

GET

URL

https://<hostname-api.sabacloud.com>/v1/report/list

Example:

https://<hostname-api.sabacloud.com>/v1/report/list?category=Compensation&audienceType=employee

Calling Options

Table 2. Calling OptionsCalling Options
Name Description Default Value Data Type Required?
audienceType Audience Type string Yes
category

If not specified, the API will return RaaS enabled reports from ALL categories.

If specified, the API will return RaaS enabled report from the SPECIFIED category.

string No

Learning Module

Parent topic: Learning

UPDATE DETAILS OF A LEARNING MODULE

Overview

Updates the details of a learning module based on the Module's ID and Certification's ID passed as parameter values.

Requires OAuth

No

Method

PUT

URL

https://<hostname-api.sabacloud.com>/v1/learningmodule/:id

Calling Options

Table 1. Calling OptionsCalling Options
Name Description Sample Value Data Type Required?
id Module's ID string Yes
certification_id Certification's ID string Yes
name Name of the learning module module modified string Yes
waiver Waiver of the learning module false string No
sequence Sequence of the learning module string No
notes Notes for the learning module note101 string No
required Whether the learning module is required true string No
customValues Custom Values string No
reqd_no_of_items The required number of items string Yes

Request Body

{
   "name": "Module B mmmoddiiffiieedd",
   "is_waiver": true,
   "sequence": 2,
   "notes": "hello yuppiiii",
   "is_reqd": true,           
   "reqd_no_of_items":1,
   "id": "cegrp000000000200049",
   "certification_id": "crtfy000000000200784
}

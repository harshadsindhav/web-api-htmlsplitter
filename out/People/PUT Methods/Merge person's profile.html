MERGE PERSON'S PROFILE

Overview

When a learner moves from one external organization to another, there is a need to move history of completed learning elements. This API merges a peron's profile.

Requires OAuth

No

Method

PUT

URL

https://<hostname-api.sabacloud.com>/v1/common/profile/:personId/action/merge

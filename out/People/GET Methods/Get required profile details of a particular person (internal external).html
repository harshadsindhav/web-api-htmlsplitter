GET REQUIRED PROFILE DETAILS OF A PARTICULAR PERSON (INTERNAL/EXTERNAL)

Overview

Returns the required profile details of a particular person based on the Person's username and type (internal or external) passed as a parameter values.

Requires OAuth

No

Method

GET

URL

https://<hostname-api.sabacloud.com>/v1/people/username=:name:(:searchFields)?type=:type

Calling Options
Table 13. Calling Options Calling Options
Name Description Sample Value Data Type Required?
name Person's user name uone string Yes
type User type [internal or external] internal string Yes

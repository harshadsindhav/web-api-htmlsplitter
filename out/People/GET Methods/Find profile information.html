FIND PROFILE INFORMATION

Overview

Returns the profile information based on certain criteria.

Requires OAuth

No

Method

GET

URL

https://<hostname-api.sabacloud.com>/v1/people?type=:type&f=(:searchFields)&q=(:query)

Calling Options
Table 1. Calling Options Calling Options
Name Description Sample Value Data Type Required?
type User type [internal or external] internal string No
searchFields Indicate the field(s) that you intend to extract: [(company_id, state, country, username, status, suffix, title, city, zip, fname, lname, email, gender, homephone, workphone, fax, ethnicity, religion, is_manager, ss_no, job_title, terminated_on, person_type, mname, started_on, job_title_type, date_of_birth, person_no, special_user, addr1, addr2, addr3, custom1, custom2, custom3, custom4, custom5, custom6, custom7, custom8, custom9, custom0, id, created_on, updated_on, is_resource] status,is_manager string No
query Search criteria [ex: status%3D%3DFull%20Time] status%3D%3DFull%20Time,is_manager%3D%3Dfalse string No

GET A CIF CERTIFICATE

Overview

Obtains a CIF certificate for a specific site based on the username and password passed as parameter values.

Method

GET

Requires OAuth

No

URL
https:// <hostname-api.sabacloud.com>/v1/coreservice/certificate/
Note:

The following is a deprecated URL for this REST Web service and will be discontinued in the next few updates:

https:// <hostname-api.sabacloud.com>/Saba/api/content/coreservice/certificate/
Calling Options
Table 1. Calling Options Calling Options
Name Description Sample Value Data Type Required?
userName The username uone string Yes
password The user password welcome string Yes
siteName The site name SabaSite string Yes
timeOutInMilliSeconds The time out in milliseconds 100000 string No

GET INITIAL SYSTEM VALUES FOR CONTENTTYPES, CONTENTFORMATS, CONTENTSERVERS

Overview

Returns the initial system values for contentTypes, contentFormats, contentServers for a particular site based the CIF certificate passed as a parameter value.

Method

GET

Requires OAuth

No

URL
https:// <hostname-api.sabacloud.com>/v1/coreservice/certificate/
Note:

The following is a deprecated URL for this REST Web service and will be discontinued in the next few updates:

https:// <hostname-api.sabacloud.com>/Saba/api/content/coreservice/certificate/
Calling Options
Table 2. Calling Options Calling Options
Name Description Sample Value Data Type Required?
certificate The CIF certificate string Yes
Note: This parameter needs to be passed as a Header Parameter.

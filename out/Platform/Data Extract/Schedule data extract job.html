SCHEDULE DATA EXTRACT JOB

Overview

Allows scheduling a data extract job.

Requires OAuth

No

Method

POST

URL

https://<hostname-api.sabacloud.com>/v1/dataextract/job/:id/scheduleJob

Calling Options
Table 2. Calling Options Calling Options
Name Description Sample Value Data Type Required?
@type @type of request body deschedule string Yes
id Data Extract Job's ID string Yes
occurenceType

Frequency of data extract.

Input Fields Frequency
occurenceType DAILY WEEKLY MONTHLY
startDate Required Required Required
endDate Required Required Required
weeklyMask null Required null
dayOfMonth 0 0 Required
[ "DAILY", "WEEKLY", "MONTHLY" ] enumerated Yes
startDate

Start Date of data extract

 {"@type":"date",

"datetime":"2016-01-22 7:30:00"}

 string Yes
endDate

End Date of data extract

 {"@type":"date",

"datetime":"2017-01-22 7:30:00"}

string Yes
dayOfMonth

Used only for monthly extract. Value can be from 1 to 31

 0 string No
weeklyMask

Used only for weekly extract. First bit represents Monday and last bit represents Sunday. Use only for weekly occurrence, otherwise the value should be null.

 0,0,0,0,0,0,0 string No

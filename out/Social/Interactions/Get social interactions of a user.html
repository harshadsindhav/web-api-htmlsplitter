GET SOCIAL INTERACTIONS OF A USER

Overview

Retrieves the social interactions such as clicks, likes, comments and shares of a user.

Requires OAuth

No

Method

GET

URL

https://<hostname-api.sabacloud.com>/v1/social/metrics/:personId?resource=(:resources)&from_date=:from_date&to_date=:to_date

URL (using lookup)

https://<hostname-api.sabacloud.com>/v1/social/metrics/username=<username>?resource=(:resources)&from_date=:from_date&to_date=:to_date

Calling Options
Table 6. Calling Options Calling Options
Name Description Default Value Data Type Required?

resources List of resources:
  • idea
  • file
  • url
  • blog
  • contributions
  • discussions
  • feedback
  • workspace

  • views
  • videos
 string Yes

string Yes
to_date To date
Note: The supported date format: YYYY-MM-DD.
string Yes
Detail of metrics fields for each resource type in output
The following are the fields for each resource type in output:
  1. Connections
    1. Total followers
    2. Total followed
    3. Grand total of followers (regardless of date range)
    4. Grand total of followed (regardless of date range)
  2. Contributions
    1. Total ideas contributed
    2. Total bookmarks contributed
    3. Total discussion initiated
    4. Total file contributed
    5. Total url contributed
    6. Total blog contributed
    7. Total video contributed
  3. Discussions
    1. Total votes for discussion
    2. Total number of replies on discussion
    3. Total number of tags on discussion
  4. Ideas
    1. Total number of tags on ideas
    2. Total number of Votes for idea
      Note: If a user gives two positive votes and two negative votes, then this value appears as 0. It is also 0, if the user does not have any votes.
    3. Total number of Views for idea
    4. Total number of comments for idea
  5. File
    1. Number of tags on file
    2. Count of Ratings for File
    3. Total comments for File
  6. URL
    1. Number of tags on URL
    2. Count of Ratings for URL
    3. Total comments for URL
  7. Blogs
    1. Number of tags on Blogs
    2. Number of Ratings for Blog
    3. Total comments for Blog
  8. Video
    1. Number of Tags on Video
    2. Count of Ratings for Video
    3. Total comments for Video
  9. Workspace
    1. Number of Tags on Workspaces
    2. Count of Ratings for Workspaces
  10. Feedback
    1. Positive votes by person
    2. Negative votes by person
  11. Views
    1. Number of Views by person for videos, page and blogpost.

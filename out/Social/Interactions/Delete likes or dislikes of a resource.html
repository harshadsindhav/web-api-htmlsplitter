DELETE LIKES OR DISLIKES OF A RESOURCE

Overview

Allows deleting likes or dislikes of a resource.

Requires OAuth

No

Method

DELETE

URL

https://<hostname-api.sabacloud.com>/v1/social/resource/:resourceId/likes

Calling Options
Table 2. Calling Options Calling Options
Name Description Default Value Data Type Required?
resourceId ID of the resource. A resource can be: Idea, Issue, Link, File and Discussion. Pages, workspaces, blogs are not supported as of now. string Yes
Return Values
{"success":true}

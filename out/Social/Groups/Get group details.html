GET GROUP DETAILS

Overview

Retrieves the details of a particular group.

Requires OAuth

No

Method

GET

URL

https://<hostname-api.sabacloud.com>/v1/social/groups/:groupId?expand=:details

Calling Options
Table 1. Calling Options Calling Options
Name Description Default Value Data Type Required?
groupId ID of the group string Yes
details Comma separated list of detail attributes string No

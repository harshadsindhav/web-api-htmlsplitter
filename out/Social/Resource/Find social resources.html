FIND SOCIAL RESOURCES

Overview

Retrieves the social resources of a given resource type. This API enables the user to search for social resources using different search criteria and different search attributes.

Requires OAuth

No

Method

GET

URL
https:// <hostname-api.sabacloud.com>/v1/social/resource?type=:resourceType
Note: The type search criteria is mandatory and will only accept one resource type at a time.
Example

https://<hostname-api.sabacloud.com>/v1/social/resource?type=idea

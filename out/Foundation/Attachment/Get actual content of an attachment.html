GET ACTUAL CONTENT OF AN ATTACHMENT

Overview

Returns the contents of the attachment based on the Attachment's ID.

Requires OAuth

No

Method

GET

URL

https://<hostname-api.sabacloud.com>/v1/attachments/attachment/:id

Note:

The following is a deprecated URL for this REST Web service and will be discontinued in the next few updates:

https:// <hostname-api.sabacloud.com>/Saba/api/platform/attachments/attachment/:id
Calling Options
Table 2. Calling Options Calling Options
Name Description Sample Value Data Type Required?
id Attachment's ID string Yes
Return Values
Renders the content of the actual attachment.

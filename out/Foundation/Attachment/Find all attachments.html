FIND ALL ATTACHMENTS

Overview

Returns all attachments for a given owner ID.

Requires OAuth

No

Method

GET

URL

https://<hostname-api.sabacloud.com>/v1/attachments?ownerName=:ownerName&ownerId=:ownerId

Note:

The following is a deprecated URL for this REST Web service and will be discontinued in the next few updates:

https:// <hostname-api.sabacloud.com>/Saba/api/platform/attachments?ownerId=:ownerId

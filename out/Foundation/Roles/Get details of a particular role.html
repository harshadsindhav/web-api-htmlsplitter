GET DETAILS OF A PARTICULAR ROLE

Overview

Returns complete information about a job role based on the Role's ID that is passed as a parameter value.

Method

GET

Requires OAuth

No

URL
https:// <hostname-api.sabacloud.com>/v1/role/:id
Note: Use ?includeassociation=true to retrieve the association details (such as owner) as well:

https://<hostname-api.sabacloud.com>/v1/role/:id?includeassociation=true

URL (User-friendly)
You can use a user-friendly URL which accepts the name instead of the internal Role's ID.

https://<hostname-api.sabacloud.com>/v1/role/name%3D<RoleName>?includeassociation=true

Calling Options
Table 2. Calling Options Calling Options
Name Description Sample Value Data Type Required?
id Role's ID string Yes
includeassociation Returns the details of associations in the search true string No

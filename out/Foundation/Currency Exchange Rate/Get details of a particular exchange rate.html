GET DETAILS OF A PARTICULAR EXCHANGE RATE

Overview

Returns complete information about a exchange rate based on the exchange rate's id that is passed as a parameter value.

Requires OAuth

No

Method

GET

URL

https://<hostname-api.sabacloud.com>/v1/exchangerate/:id:(:searchFields)

Example:

https://<hostname-api.sabacloud.com>/v1/exchangerate/ccyex000000000200001

Calling Options
Table 3. Calling Options Calling Options
Name Description Default Value Data Type Required?
id Exchange Rate's ID string Yes
searchFields Indicate what additional details needs to be returned. Acceptable values are (case-sensitive) string No

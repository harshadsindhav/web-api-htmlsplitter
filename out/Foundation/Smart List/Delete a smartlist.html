DELETE A SMARTLIST

Overview
Allows deleting an existing smartlist that is not associated to other objects based on the Smartlist ID.
Note: Only users with the People admin privilege will be able to create, update or delete Smarlists using these APIs.
Requires OAuth

No

Method

DELETE

URL

https://<hostname-api.sabacloud.com>/v1/smartlists/:id

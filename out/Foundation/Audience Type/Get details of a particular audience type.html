GET DETAILS OF A PARTICULAR AUDIENCE TYPE

Overview

Returns complete information about an Audience Type based on the Audience Type's ID that is passed as a parameter value.

Requires OAuth

No

Method

GET

URL

https://<hostname-api.sabacloud.com>/v1/audiencetype/:id

URL (User-friendly)
You can use a user-friendly URL which accepts the name instead of the internal Audience Type's ID.

https://<hostname-api.sabacloud.com>/v1/audiencetype/name%3D<AudienceTypesName>

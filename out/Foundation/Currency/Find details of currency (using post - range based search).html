FIND DETAILS OF CURRENCY (USING POST - RANGE BASED SEARCH)

Overview

Returns the details of the currency along with the ID, Name and the Deeplink URL based on the provided search criteria.

Requires OAuth

No

Method

POST

URL

https://<hostname-api.sabacloud.com>/v1/currency/searchQuery?count=:count&startPage=:startPage&includeDetails=:includeDetails

Example:

https://<hostname-api.sabacloud.com>/v1/currency/searchQuery?count=10&startPage=1

Calling Options
Table 5. Calling Options Calling Options

conditions Conditions

\"operator\": \"==\",
\"value\": \"USD\"}]

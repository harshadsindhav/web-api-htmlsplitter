GET DETAILS OF A PARTICULAR SUB AUDIENCE TYPE

Overview

Returns complete information about a sub audience type based on the sub audience type's ID that is passed as a parameter value.

Requires OAuth

No

Method

GET

URL

https://<hostname-api.sabacloud.com>/v1/subaudiencetype/:id:(:searchFields)

Calling Options
Table 4. Calling Options Calling Options
Name Description Default Value Data Type Required?
id Sub audience type's ID string Yes
searchFields Indicate what additional details needs to be returned. Acceptable values are (case-sensitive) string No

EXECUTE ASYNC REPORT

Overview

This API will fetch the data of the report which was executed in the async mode

Requires OAuth

No

Method

POST

URL

https://<hostname-api.sabacloud.com>/v1/report/execute/:reportId/:chartId

Calling Options
Table 4. Calling Options Calling Options
Name Description Default Value Data Type Required?
reportId Report ID.

Starts with: ‘rptdf’

 string Yes
chartId Chart ID.

Starts with: ‘rppsd’

 string Yes
reportRunId Report run id (Required to excute an async report)

Mandatory: No (for synchronous execution)

Mandatory: Yes (for asynchronous execution)

string Yes

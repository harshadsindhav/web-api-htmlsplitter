EXECUTE REPORT

Overview

This API will execute the report in the synchronous / asynchronous (depending upon the configuration)

Requires OAuth

No

Method

POST

URL

https://<hostname-api.sabacloud.com>/v1/report/execute/:reportId/:chartId

Exmaple:

https:// <hostname-api.sabacloud.com>/v1/report/execute/rptdf0000007904c03f0156a16c63e5008000/rppsd0000005d584d490156b0e88d5d007fe8
Calling Options
Table 5. Calling Options Calling Options
Name Description Default Value Data Type Required?
reportId Report ID.

Starts with: ‘rptdf’

 string Yes
chartId Chart ID.

Starts with: ‘rppsd’

 string Yes
audienceType Audience Type

If not specified, the API will execute the report definition with respect to ADMIN audience type.

If specified, the API will execute the report definition if the report is shared with the specified audience type.

Refer: #ENUM for supported audience type.

string Yes
@type Execution parameters com.saba.services.rest.dto. APIExecutionDTO string Yes
pageSize Page size.

Mandatory: Yes (for synchronous execution)

Mandatory: No (for asynchronous execution)

20 string Yes
pageNumber Page number

Mandatory: Yes (for synchronous execution)

Mandatory: No (for asynchronous execution)

 1 string Yes
filter Filter object

Mandatory: Yes (for synchronous execution)

Mandatory: No (for asynchronous execution)

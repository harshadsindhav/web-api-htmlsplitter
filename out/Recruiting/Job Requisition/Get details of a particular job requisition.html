GET DETAILS OF A PARTICULAR JOB REQUISITION

Overview

Returns the details of a particular job requisition.

Requires OAuth

No

Method

GET

URL

https://<hostname-api.sabacloud.com>/v1/job-requisition/:id:(:searchFields)

Calling Options
Table 4. Calling Options Calling Options
Name Description Default Value Data Type Required?
id The ID of the desired job requisition jobrq000000000001020 string Yes
searchFields Indicate what additional details need to be returned in the output. Acceptable values are (case-sensitive)

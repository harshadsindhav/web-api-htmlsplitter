SEARCH PURPOSE LIST

Overview

Retreives a list of purpose for the given type: person/equipments/inventory/classrooms.

Requires OAuth

No

Method

GET

URL

https://<hostname-api.sabacloud.com>/v1/learning/resource/purpose?type=:resourcetype&count=:count&startPage=:startPage

Calling Options
Table 4. Calling Options Calling Options
Name Description Default Value Data Type Required?
resourcetype The name of the desired component. person enumerated Yes
count The number of records per page. 10 string No
startPage The start page number for the list of records. 1 string No

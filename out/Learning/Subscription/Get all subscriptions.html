GET ALL SUBSCRIPTIONS

Overview

Returns all Subscriptions for which the user has access.

Requires OAuth

No

Method

GET

URL

https://<hostname-api.sabacloud.com>/v1/subscription?count=:count&startPage=:startPage

Calling Options
Table 3. Calling Options Calling Options
Name Description Sample Value Data Type Required?
count The number of records per page. 10 string No
startPage The start page number for the list of records. 1 string No

FIND DETAILS OF CERTIFICATION/CURRICULUM

Overview

Returns the details of the Certification/Curriculum along with the ID, Name and the Deeplink URL based on the provided search criteria.

Requires OAuth

No

Method

GET

URL

https://<hostname-api.sabacloud.com>/v1/:componentName?q=(:criteria_field==:field_value)&count=:count&startPage=:startPage

Calling Options
Table 2. Calling Options Calling Options
Name Description Sample Value Data Type Required?
componentName The name of the desired component (certification or curriculum). curriculum enumerated Yes
criteria_field The search criteria i.e. the field name.
You can use any field with "isSearchFilter": true.
Note: To find out the fields that have "isSearchFilter": true, invoke the "Get the Meta details" REST APIs.
name string Yes
field_value The search value for the specified search criteria. abc string Yes
count The number of records per page. 10 string No
startPage The start page number for the list of records. 1 string No

APPROVE / REJECT AN ORDER ITEM

Overview

Allows approving / rejecting an Order Item.

Requires OAuth

No

Method

PUT

URL

https://<hostname-api.sabacloud.com>/v1/learning/enroll/:regId?action=:action

Note: The people admin can approve or reject an order item by using the following URL that holds an additional parameter overrideApprovalChain=true:

https://<hostname-api.sabacloud.com>/v1/learning/enroll/regId?overrideApprovalChain=true&action=:action

Calling Options
Table 3. Calling Options Calling Options
Name Description Sample Value Data Type Required?
regId Registration's ID regdw000000000004020 string Yes
action The action to be performed. Can be approve or reject. approve string Yes

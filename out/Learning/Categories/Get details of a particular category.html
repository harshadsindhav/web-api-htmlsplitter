GET DETAILS OF A PARTICULAR CATEGORY

Overview

Returns complete information about a category based on the category's ID that is passed as a parameter value.

Requires OAuth

No

Method

GET

URL

https://<hostname-api.sabacloud.com>/v1/learning/category/:id

URL (User-friendly)
You can use a user-friendly URL which accepts the name instead of the internal Category's ID.

https://<hostname-api.sabacloud.com>/v1/learning/category/name%3D<CategoryName>

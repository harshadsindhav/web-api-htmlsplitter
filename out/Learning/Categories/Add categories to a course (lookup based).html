ADD CATEGORIES TO A COURSE (LOOKUP BASED)

Overview
Adds categories to a course and returns a reference to the course to which the Categories were added.
Note: Using this API you can list the categories simply by using lookup i.e. "displayName" and "id" is not required.
Requires OAuth

No

Method

POST

URL

https://<hostname-api.sabacloud.com>/v1/learning/category/course/:courseid

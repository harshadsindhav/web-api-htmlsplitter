GET META DETAILS OF A COMPONENT

Overview

Returns the meta details of a particular component [certification | curriculum | path | learningmodule | learningintervention].

Requires OAuth

No

Method

GET

URL

https://<hostname-api.sabacloud.com>/v1/:name/meta

Calling Options
Table 3. Calling Options Calling Options
Name Description Sample Value Data Type Required?
name The name of the component. Can be one of these:
  • certification
  • curriculum
  • path
  • learningmodule
  • learningintervention
 certification string Yes

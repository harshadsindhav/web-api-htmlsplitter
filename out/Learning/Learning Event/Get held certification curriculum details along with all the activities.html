GET HELD CERTIFICATION/CURRICULUM DETAILS ALONG WITH ALL THE ACTIVITIES

Overview

Gets held Certification/Curriculum details along with all the activities

Requires OAuth

No

Method

GET

URL

http://<hostname-api.sabacloud.com>/v1/learning/heldlearningevent/:heldCertId

Calling Options
Table 3. Calling Options Calling Options
Name Description Sample Value Data Type Required?
heldCertId Held Certification/Curriculum's ID string Yes

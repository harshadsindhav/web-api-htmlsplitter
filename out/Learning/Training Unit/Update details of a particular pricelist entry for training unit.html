UPDATE DETAILS OF A PARTICULAR PRICELIST ENTRY FOR TRAINING UNIT

Overview

Updates a pricelist entry for the training unit based on the Price list entry's ID and Training Unit's ID that are passed as parameter values.

Requires OAuth

No

Method

PUT

URL

https://<hostname-api.sabacloud.com>/v1/trainingunit/:tuid/priceList/:plid

Calling Options
Table 12. Calling Options Calling Options
Name Description Sample Value Data Type Required?
tuid TrainingUnit's ID string Yes
plid Price list entry's ID string Yes
curr_id Currency for pricelist

Supports lookup (Currency).

 
{"id": "crncy000000000000167", "displayName": "US Dollars"}
 string Yes
amount Amount per training unit 10 string Yes

{
            "curr_id": {
                "id": "crncy000000000002353",
                "displayName": null
            },
            "amount":45

CREATE A NEW TRAINING UNIT

Overview

Allows creating a new training unit.

Requires OAuth

No

Method

POST

URL

https://<hostname-api.sabacloud.com>/v1/trainingunit

Calling Options

  • AllCurrenciesInSystem

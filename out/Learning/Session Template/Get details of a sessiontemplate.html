GET DETAILS OF A SESSIONTEMPLATE

Overview

Returns complete information about a Session Template based on the Session Template's ID that is passed as a parameter value.

Requires OAuth

No

Method

GET

URL

https://<hostname-api.sabacloud.com>/v1/sessiontemplate/:id:(:searchFields)

URL (User-friendly)
You can use a user-friendly URL which accepts the agreement_no instead of the internal TrainingUnitAgreement's ID.

https://<hostname-api.sabacloud.com>/v1/sessiontemplate/name=<SessionTemplateName>

Note: The SessionTemplate API does not contain a body attribute which can hold the ID and hence lookup is not required in the Request body.

CANCEL AN ENROLLMENT

Overview

Allows canceling an enrollment. In addition to cancelling an enrollment, this API also takes care of refunding the charges, if any according to drop policy.

Requires OAuth

No

Method

PUT

URL

https://<hostname-api.sabacloud.com>/v1/learning/enroll/:regId?action=cancel

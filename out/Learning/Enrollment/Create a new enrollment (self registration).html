CREATE A NEW ENROLLMENT (Self registration)

Overview
Creates a new enrollment without the payment option. This API is purely for self registration.
Note: Use ENROLL FOR OTHER USERS/SELF to create enrollments for other users as well as for self.

While registering by default ignoreWarnings is set to true. Setting it to false will not allowing registering.

Requires OAuth

No

Method

POST

URL

https://<hostname-api.sabacloud.com>/v1/enrollments

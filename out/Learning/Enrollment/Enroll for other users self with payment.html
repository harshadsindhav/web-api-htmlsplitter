ENROLL FOR OTHER USERS/SELF WITH PAYMENT

Overview
Allows enrolling for other users as well as for self and paying either using a Purchase Order or Training Units.
Note: This API does everything that the ENROLL FOR OTHER USERS/SELF API does and also supports payment. It works even if the payment option is not provided.
Requires OAuth

No

Method

POST

URL

https://<hostname-api.sabacloud.com>/v1/enroll

GET EXTENDED DETAILS OF A PARTICULAR REVIEW CYCLE

Overview

Returns extended details of a review-cycle.

Requires OAuth

No

Method

GET

URL

https://<hostname-api.sabacloud.com>/Saba/api/performance/reviews/reviewCycle/:id?expand=:searchFields

Calling Options
Table 8. Calling Options Calling Options
Name Description Default Value Data Type Required?
id The ID of the desired review cycle rcycl000000000001280 string Yes
searchFields Indicate what additional details needs to be returned. Acceptable values are (case-sensitive) reviewcycle,ratinglevels,requiredraters,orgratinglevels,reviewform reviewcycle,ratinglevels string No

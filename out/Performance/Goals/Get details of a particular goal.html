GET DETAILS OF A PARTICULAR GOAL

Overview

Returns complete information about a goal based on the goal assignment ID that is passed as a parameter value.

Requires OAuth

No

Method

GET

URL

https://<hostname-api.sabacloud.com>/v1/performance/goals/:id

Calling Options
Table 2. Calling Options Calling Options
Name Description Default Value Data Type Required?
id Goal ID gasgn000000000001336 string Yes
searchFields Indicate what additional details needs to be returned: progress, progress_history, approvers, parent, subgoals, tasks, changelog. Acceptable values are (case-sensitive) string No
